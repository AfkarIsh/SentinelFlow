# SentinelFlow
SentinelFlow is a Multi-Agent SOC Incident Response system built with Google ADK &amp; Gemini 1.5 Flash. It automates cyber threat triage by sequentially orchestrating LogInvestigator, ThreatReasoner, and ReportReviewer agents to analyze logs, reconstruct timelines, map MITRE ATT&amp;CK tactics, and output a validated mitigation blueprint.
